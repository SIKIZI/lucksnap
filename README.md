# Lucksnap - Lottery Number Generator and Simulator

This is a simple web application that allows you to generate random lottery numbers and simulate a lottery drawing.

## Features

- **Generate Lottery Numbers:** Get a set of 6 unique random numbers between 1 and 45.
- **Simulate Lottery Drawing:** Input up to 5 sets of your own lottery numbers and see if you win against a random drawing.

## How to Use

1.  **Generate Numbers:** Click the "새 번호 생성" (Generate New Numbers) button to get a new set of lottery numbers.
2.  **Simulate Drawing:**
    -   Enter your lottery numbers in the input fields. Each set of 6 numbers should be separated by commas.
    -   Click the "추첨 시작" (Start Drawing) button.
    -   The app will display the winning numbers and the results for each of your sets.

## Design

The app is designed to be simple and user-friendly, with a clean interface that is easy for people of all ages to use.
